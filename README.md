
# ArianeeJS


## Getting Started

 
### Get ready : installation
  
  Get the last version from NPM or your favorite registry:

````
npm i @arianee/arianeejs -D
````

### Simple example

```
import {Arianee} from "arianeeJS";

Arianee()
    .fromRandomKey();
```
